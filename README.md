# ALX Listing App

A modern Airbnb clone built with Next.js, TypeScript, and TailwindCSS.

## Project Overview
This project aims to create a property listing platform similar to Airbnb, focusing on clean design, responsive layouts, and reusable components.

## Project Structure
- `components/` - Reusable React components
  - `common/` - Shared components (Card, Button)
- `interfaces/` - TypeScript type definitions and interfaces
- `constants/` - Application constants and configuration
- `public/assets/` - Images, icons, and static files
- `pages/` - Next.js page components
- `styles/` - Global CSS and Tailwind imports

## Technologies Used
- Next.js 13+ (Pages Router)
- TypeScript
- TailwindCSS
- ESLint
